D - jeżeli występuje serwer to tak jak w punke C, na egzaminie jest losowaość i dostępne są 2 rodzaje oprogramowania (Linux Ubuntu Server, Lub Microsoft Server 2022 )

D1 - rozwiąż w wersji gdyby był to system Windows
D2 - równolegle rozwiąż te same zadania co w punkcie D1, ale dla Ubuntu Server
Wskaż też jak odczytać dane skonfigurowane ustawienia, jakich uzyć komend, żeby je wyświetlić i mieć do nich szybki dostęp dla D1 i D2.

Zadania inne niż sieciowe na przykład kosztorysy itp, postaraj się wskazać dobre rozwiązania, reguły, wyniki

Rozwiązania przedstawiaj w sposób analityczny, szczegółowy, opisowy i wyjaśniający
